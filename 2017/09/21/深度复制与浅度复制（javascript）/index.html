为什么会产生浅复制这种现象？

对于对象来说，浅复制是对对象地址的复制，没有开辟新的栈，
也就是复制的结果是两个对象指向同一个地址，修改其中一个对象的属性，则另一个对象的属性也会改变；

那么怎么解决呢？

此方法的缺点是：

  • 只能处理Number, String, Boolean, Array等对象
